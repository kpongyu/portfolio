Mobile Application redesign

My Role: Interaction Designer, Visual Designer
Duration: 3 Months

Current Design solution:
Inconsistent and lack of empathy

There are several design challenges in current application design:

First, the visual branding doesn't align with the theme, the gears in big head looks scary and misleading, without any linkage with the function of improving sleeping patterns.
Second, there is no inconsistant design pattern between different pages.
Finally, the application is somewhat text-heavy, it is hard to describe the context like "ocean view", "fun" just in text.

First Version of Re-design: The metaphor of starry night

I use the metaphor of starry night to create a atmosphere of dream and peacefulness. I chose purple as the theme color with tint images and blurry gradients.
I conducted several user testings on the first version, the major issue comes from the information flow, there are too much steps for users to reach to the desired places and there are spaces for rearchitect.

Second Version of Re-design: Emphasis the content in a more integrated way

Based on the discussion, I implement the design in a more integrated way: I used a grid layout to represent sound tracks in different categories, when user listen to a specific soundscope, the image of soundscope would expand to cover the full screen to create the context.I also used moon as the metaphor to keep track of users' sleeping condition.
I built a Framer prototype for user testings, you can also interact with the prototype on the first section of the page.
